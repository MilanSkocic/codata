NAME
       codatacli(1) - Command line for codata

SYNOPSIS
       codatacli [OPTIONS]

DESCRIPTION
       codatacli  is command line interface which prints all the codata
       constants.  The current values are from 2022.

OPTIONS
       o --year, -y
              Year of the codata constants: 2022, 2018, 2014, 2010

       o --help
              Show help text and exit

       o --verbose
              Display additional information when available.

       o --version
              Show version information and exit.
