\doxysubsection{Detailed Description}
Source code generator for C and Fortran languages. 

The raw codata from \href{http://physics.nist.gov/constants}{\texttt{ http\+://physics.\+nist.\+gov/constants}} are converted into C and Fortran code. The generator generates\+:
\begin{DoxyItemize}
\item an module for Fortran
\item and header and source files for C The generated files are then compiled into shared and static libraries. 
\end{DoxyItemize}
