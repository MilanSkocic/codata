 Simple program for generating codata for C, Fortran.

The raw codata from \href{http://physics.nist.gov/constants}{\texttt{ http\+://physics.\+nist.\+gov/constants}} are converted into C and Fortran code. The generator generates\+:
\begin{DoxyItemize}
\item an module for Fortran
\item and header and source files for C
\end{DoxyItemize}

The generated files are then compiled into shared and static libraries\+:
\begin{DoxyItemize}
\item libccodata for C
\item libfcodata for Fortran
\end{DoxyItemize}

The libraries for C and Fortran can be installed in order to be included in programs.
