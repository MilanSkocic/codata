codata¶

codata provides the lastest codata constants (2018). The raw codata are taken from http://physics.nist.gov/constants.

How to install¶

An Makefile is provided which uses fpm for building the library with additional options:

  • compile the source generator and generate the sources

  • copy the library into the python wrapper folder

  • install the C headers

  • uninstall the library and headers

On windows, msys2 needs to be installed and use the mingw64 or mingw32 terminals.

Build

source configuration
make

Run tests

fpm test

Install

make install

Uninstall

make uninstall

Then, if you need to build to Python wrapper from sources

cd pywrapper
python setup.py build_ext --inplace

Dependencies¶

gcc>=9
gfortran>=9
