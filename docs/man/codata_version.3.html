NAME

version - function

LIBRARY

Codata library (libcodata, -lcodata)

SYNOPSIS

Fortran

character(len=:), pointer :: version()

C

char* codata_version()

Python

pycodata.__version__

DESCRIPTION

Get the library version.

EXAMPLE

Fortran:
use codata
print *, "version = ", version()

C:
include <stdio.h>
include "codata.h"
printf("version = %s\n", codata_version());

Python:
import pycodata
print(f"version = {pycodata.__version__}")
