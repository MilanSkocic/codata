NAME
       get_version - get the version of the library

LIBRARY
       codata (-libcodata, -lcodata)

SYNOPSIS
       get_version()

DESCRIPTION
       Get the library version.

       Deprecated.  It  will  be  removed in the next major release 3.0
       when the new codata constants will be released (2026).  Use  co‐
       data_version(3) instead.

RETURN VALUE
       character(len=:), pointer :: fptr
              Pointer to a string.

EXAMPLE
       Minimal example:

             use codata
             implicit none

SEE ALSO
       codata_version(3)
